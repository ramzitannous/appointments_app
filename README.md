**Appointments Mobile APP**

add appointments and save them to firebase Realtime db

**Libraries**
1) react-navigation
2) @reduxjs/toolkit
3) styled-components
4) react-native-calendars
5) react-native-date-picker
6) @react-native-firebase

** Note: only tested on Android
